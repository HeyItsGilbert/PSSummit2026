## Why Static Sites?

Static sites are:

1. **Fast** - Pre-rendered HTML, no server-side processing
2. **Secure** - No database, no attack surface
3. **Free to host** - GitHub Pages, Netlify, Cloudflare Pages
4. **Written in Markdown** - Which you *already know*
